# About Me Website

This project is a website that demonstrates the development skills I've learned as a Software Developer.

To run page locally:
1. Clone repo: https://github.com/katedakins/aboutme.git.
2. Run http-server in terminal.
3. Open local host at http://localhost:8080/.

To view page in production:
1. Commit changes.
2. Push changes.
3. Publish changes.
4. Go to: https://katedakins.github.io/aboutme/.
